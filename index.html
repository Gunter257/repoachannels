Gunter25 Addons Repository

Aquí puedes encontrar el archivo ZIP para instalar el repositorio en Kodi:

  • repository.acestream-channels.zip
  • addons.xml
  • addons.xml.md5

Para instalar el addon, primero añade esta URL como fuente en Kodi y luego busca 'repository.acestream-channels.zip' para instalar el repositorio.

Una vez instalado el repositorio, podrás instalar el addon 'Acestream Channels' desde la sección 'Instalar desde repositorio'.
